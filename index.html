A電腦規格

  • 作業系統
    macOS 12 以上 / Windows 10 以上 / Ubuntu 22.04 以上都可以，但 macOS 與 Linux 跑 Claude Code 體驗最順
  • 記憶體
    8GB 起跳，建議 16GB記憶體小一點會跑 dev server 卡頓
  • 硬碟空間
    至少 15GB 可用會裝 Node.js、Git、VS Code、各種專案
  • 瀏覽器
    Chrome 最新版很多服務（Supabase / Zeabur）對 Chrome 支援最完整
  • 編輯器
    VS Code（免費）當天會在現場一起裝
  • 網路
    穩定 Wi-Fi（會場提供）建議自己也準備手機熱點備用

常見問題

你可能在想的。

我完全不會寫程式，可以來嗎？

完全可以——而且這堂課就是為你設計的。如果你已經會寫程式，反而會覺得節奏太慢。半日班只看不做，一日班會做但 Claude 幫你寫，你的工作是「描述、判斷、決定」。三個月不會程式的人在這堂課做出作品的比會程式的工程師還快，因為他們沒有舊習慣需要拋棄。

半日班為什麼免費？是要推銷什麼嗎？

誠實說：半日班是設計來讓你決定要不要報一日班的。我不會強迫推銷，但會在課程結束時告訴你一日班還剩幾個名額。如果你只想看半日班、回家自己摸索——完全 OK，課後資源包還是會給你。

一日班 NT$6,800 真的值得嗎？

包含 Claude Pro 一個月（NT$650）、午餐、課後 30 天 LINE 支援、課程錄影、Notion 資源包。換算下來實際課程費用大約 NT$5,500，平均每小時 NT$800。對比一張認證的證照課動輒 NT$15,000+ 你還是學不會實作——我覺得很划算。但如果你不確定，先來免費的半日班看看。

我的筆電很舊跑得動嗎？

只要能打開 Chrome、跑 VS Code 不卡，基本上都能跑。Claude Code 的運算在雲端，本機不需要強。真的擔心可以報名前傳一張筆電型號給我看，我幫你確認。

我可以在課堂上做我自己的工作專案嗎？

一日班可以。報名後會請你填一份小問卷，描述你想做的題目，我會在課前準備好對應的範例 prompt 跟流程。當天你就是做你自己的東西，上線之後就是你公司在用的工具。這是很多學員最大的驚喜。

課後我跟不上、卡住怎麼辦？

30 天的 LINE 群組就是為這個設計的。你可以丟錯誤訊息、丟截圖、丟程式碼，我跟其他學員都會幫忙。如果是企業客戶，我也提供額外的顧問服務（另計）。

有線上版嗎？

2026 年下半年會開直播線上版，目前以實體班為主——因為小班制的一對一品質是線上很難複製的。半日班我們會錄影並剪成 YouTube 系列，當作未來線上課的引流。

可以企業包班嗎？

可以。8 人以上開團、企業內訓、產業特化版本都接。最近做過：水域運動產業 AI 導入、SEO 公司 GEO 工作坊、行銷公司 AI 內容工作流。下方 CTA 填表選「企業內訓」我會直接聯繫。
